# Globo Test - API

## Project setup
```
yarn
```

### Compiles and serve for production
```
yarn start
```

### Compiles and hot-reloads for development
```
yarn start:local
```

### Compiles and minifies for production
```
yarn build
```

### Lints files
```
yarn lint
```

### Lints and fix files
```
yarn lint:fix
```

### Check the coding style of the files
```
yarn style:check
```

### Run functional tests
```
yarn test:functional
```

### Run unit tests
```
yarn test:unit
```
